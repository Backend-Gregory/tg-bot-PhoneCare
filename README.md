# 📱 Телеграм-бот для мастерской по ремонту телефонов PhoneCare

Бот-визитка с расширенной записью на услуги.

## 🚀 Функции

### Визитка
- 📞 Контакты
- 💼 Услуги
- 💰 Цены
- 📍 Адрес
- ❓ Помощь

### Запись на ремонт (расширенная)
- Выбор услуги (Замена экрана, аккумулятора, диагностика, ремонт кнопок)
- Выбор мастера (зависит от выбранной услуги)
- Сбор имени, телефона и удобного времени
- Отправка полной заявки владельцу в Telegram

## 🛠 Технологии

- Python 3.11
- aiogram 3.x
- FSM (машина состояний)
