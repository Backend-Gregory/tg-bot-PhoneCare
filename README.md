# Телеграм-бот для мастерской по ремонту телефонов

Бот-визитка с записью на услуги.

## Функционал

- 📞 Контакты
- 💼 Услуги
- 💰 Цены
- 📍 Адрес
- ❓ Помощь
- 📝 Записаться (опрос: имя → телефон)

## Команды

- `/start` — главное меню
- `/cancel` — отмена записи

## Технологии

- Python 3.10
- aiogram 3.x
- FSM (машина состояний)
